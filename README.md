# PulseGPT
A 125M parameter GPT-style medical language model trained from scratch on 85M tokens of PubMed research, capable of generating coherent clinical text

## What it does
- Medical Q&A
- Clinical notes summarisation

## Built with
- nanoGPT (Karpathy)
- PyTorch + Apple MPS (M5 Pro, 48GB RAM)
- 85M tokens of PubMed medical text

## Training
Trained from scratch on a MacBook Pro M5 Pro.
