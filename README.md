# PulseGPT
A 125M parameter GPT-style language model trained from scratch on PubMed medical abstracts.

## What it does
- Medical Q&A
- Clinical notes summarisation

## Built with
- nanoGPT (Karpathy)
- PyTorch + Apple MPS (M5 Pro, 48GB RAM)
- 85M tokens of PubMed medical text

## Training
Trained from scratch on a MacBook Pro M5 Pro.
